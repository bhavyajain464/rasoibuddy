# Onboarding staple thumbnails

WebP icons (128×128) shown in onboarding step “Kitchen Staples”.

## Replace an image

Drop a new `{id}.webp` here. The filename must match `id` in `src/data/onboardingStaples.ts`.

Recommended: **128×128** WebP, quality **80–85**, soft cream background `#F3F4F2` to match the set.
